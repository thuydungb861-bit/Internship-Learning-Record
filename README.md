# Imu-Learning

# 🚀 HWT906P ROS 2 驱动 

> 这是一个专为维特智能 (WitMotion) HWT906P 惯性导航模块编写的 ROS 2 Python 驱动包。本项目包含底层协议的高效解析器与 ROS 2 节点封装，能在 Jetson Orin NX 等高性能平台上稳定、高速地输出 IMU 姿态数据。

## ✨ 核心特性

本驱动针对 HWT906P 的发包物理特性进行了深度定制，主要包含两大核心逻辑：

1. **🛡️ 严苛的底层协议解析器 (Parser)**
   - **精确切帧与校验**：严格按照 11 字节/帧的规矩处理串口字节流，内置滑动窗口寻找帧头 (`0x55`) 与 Checksum 校验，自动丢弃干扰/脏数据。
   - **时序状态机组装**：基于维特 IMU 固定的发包顺序（`先发加速度 -> 再发角速度 -> 最后发角度/四元数`），设计了暂存草稿箱 (`PendingState`)。只有当一整轮数据完美凑齐后，才会打包成一个完整的 `ImuSample` 输出，彻底杜绝数据残缺导致的姿态解算跳变。

2. **⚡ 高性能 ROS 2 驱动节点 (Node)**
   - **大块数据吞吐**：采用一次性读取 4096 字节的高效策略，极大地降低了 Jetson Orin NX 上的 CPU 串口中断开销。
   - **自适应波特率侦测**：节点启动或通信中断时，会自动轮询并尝试匹配 IMU 的真实波特率，实现“插上就能用”的开箱即用体验。
   - **标准 ROS 接口**：将解析后的物理量转换至 ROS 标准单位 ($m/s^2$, $rad/s$)，并实时发布至标准的 `sensor_msgs/msg/Imu` 话题。
